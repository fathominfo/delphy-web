Formatting for Sequence files

Delphy can accept FASTA and MAPLE files. They need to be aligned, and the metadata line should be  |  delimited with the date in the last position. Dates should be formatted  YYYY-MM-DD .

The unique identifier for the sequence should be in the first position.

Metadata

Metadata can be loaded in a  csv  or  tsv  file. It should have one header row with column names. After the header, there should be one row per sequence. There must be one column named  id  containing the unique identifier for the sequence.
