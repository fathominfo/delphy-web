Formatting for FASTA files

FASTA files need to be aligned, and the metadata line (after the  > ) should be  |  delimited with the date in the last position. The date should be formatted  YYYY-MM-DD .

The unique identifier for the sequence should be in the first position.

Metadata

Metadata can be uploaded in a  csv  or  tsv  file. It should have one header row with column names. After the header, there should be one row per sequence. There must be one column named  id , and it should contain the unique identifier for the sequence.
